### 📋선수 정보 추가

#### 요청

| 업무          | 선수 정보 추가                                               |
| ------------- | ------------------------------------------------------------ |
| URL           | /players                                                      |
| 요청 메소드   | POST                                                         |
| 콘텐트 타입   | application/json                                             |
| 메세지 구조   | - **player : 선수이름**<br />- **team : 팀이름**<br />- age : 선수 나이<br />- nickname : 선수 별명 |
| 요청메세지 예 | { <br/> "player" : "네이마르 다 실바",<br/> "team" : "파리생제르맹",<br/> "age" : "1992",<br/> "nickname" : "차기 축구황제"<br/>} |

#### 응답

| 컨텐트 타입 | JSON                                                         |
| ----------- | ------------------------------------------------------------ |
| 메세지 구조 | - msg : 성공/실패 메세지<br />- data<br />-- id : 선수 ID<br />-- player : 선수이름<br />-- team : 팀이름<br />-- age : 선수 나이<br />-- nickname : 선수 별명 |
| 메세지 예   | {<br/>    "msg": "SUCCESS : 다음 내용을 추가하였습니다.",<br/>    "data": {<br/>        "id": 2,<br/>        "player": "네이마르 다 실바",<br/>        "team": "파리생제르맹",<br/>        "age": "1992",<br/>        "nickname": "차기 축구황제"<br/>    }<br/>} |
